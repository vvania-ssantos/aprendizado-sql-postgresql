# 💡 Dicas de Quem Está Aprendendo SQL

Esse arquivo é um conjunto de anotações pessoais, dicas práticas e aprendizados que fui tendo durante meus estudos com SQL (PostgreSQL). Se você também está começando, espero que isso te ajude 💜

---

## 📌 O que eu gostaria de ter ouvido no começo:

- **Comece devagar:** Tentar aprender tudo de uma vez só vai te deixar frustrada.
- **SELECT não é só “pegar dados”** — é a base de tudo, então estude com calma.
- **Não pule o WHERE!** Vai por mim: a maior parte dos erros estão ali.
- **Não tenha medo de apagar a tabela errada — tenha backups!** 😅
- **Testar comandos no pgAdmin ou DBeaver ajuda muito.**

---

## 🧠 Minhas estratégias que funcionaram:

- Escrever os comandos **à mão no caderno** antes de digitar
- Criar um mini banco de dados fictício pra treinar (usei loja virtual!)
- Fazer desafios do tipo: “Como eu buscaria só os clientes de SP com pedidos acima de 500?”
- Documentar tudo no GitHub me fez fixar o conteúdo e ainda ajuda outras pessoas
